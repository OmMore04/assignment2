# assignment2
A bookstore project is a system designed to manage and sell books, allowing users to browse, search, and purchase books easily. It can include features like inventory management, customer records, and online ordering.

## Live Demo
Check out the live application here: [https://assignment2-production-af4e.up.railway.app/](https://assignment2-production-af4e.up.railway.app/)

## Tech Stack
- **Frontend**: React.js, Tailwind CSS, DaisyUI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Atlas)
- **Deployment**: Railway
